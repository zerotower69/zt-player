# web音乐播放器

> 基于vue开发的音乐播放器组件，几行代码即可快速引用到自己的项目当中。

## 使用

```shell
npm i zt-player
```

## 开发

使用

```shell
npm run build
```

将修改的项目重新编译后再浏览器打开examples下的index.html文件，如果你使用VSCode，可以使用其Live Server功能来检验项目。
